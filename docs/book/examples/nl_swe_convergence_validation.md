# Example: NL-SWE Convergence Validation

**Crate**: `kwavers`  
**Run**: `cargo run -p kwavers --example nl_swe_convergence_validation`  
**Source**: [`crates/kwavers/examples/nl_swe_convergence_validation.rs`](../../../crates/kwavers/examples/nl_swe_convergence_validation.rs)

## Overview

Convergence validation for the Nonlinear Shear Wave Elastography (NL-SWE) solver against known analytical solutions. Demonstrates the convergence testing framework.

## What it tests

- Analytical convergence studies (spatial and temporal refinement)
- Convergence rate measurement vs. expected order
- Comparison against manufactured solutions
