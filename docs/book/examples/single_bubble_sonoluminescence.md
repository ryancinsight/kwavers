# Example: Single Bubble Sonoluminescence

**Crate**: `kwavers`  
**Run**: `cargo run -p kwavers --example single_bubble_sonoluminescence`  
**Source**: [`crates/kwavers/examples/single_bubble_sonoluminescence.rs`](../../../crates/kwavers/examples/single_bubble_sonoluminescence.rs)

## What This Example Demonstrates

This example models a single collapsing bubble and compares bremsstrahlung and Cherenkov-style emission signatures generated by extreme cavitation conditions. It is a compact demonstration of how bubble dynamics and optical emission models are coupled.

| Component | API | Value |
|---|---|---|
| Bubble setup | `BubbleParameters` | Initializes a 10 µm bubble in an argon-like gas state |
| Collapse model | `KellerMiksisModel` | Propagates the bubble wall dynamics under acoustic driving |
| Emission model | `IntegratedSonoluminescence` | Evaluates light emission hypotheses on top of the collapse history |

## Key Code Snippet

```rust
let bubble_params = BubbleParameters {
    r0: 10e-6,                      // 10 μm initial radius
    t0: 300.0,                      // 300 K ambient temperature
    gamma: 1.4,                     // Air polytropic index
    initial_gas_pressure: 101325.0, // 1 atm
    ..Default::default()
};

// Run comprehensive sonoluminescence simulation
let results = run_comprehensive_simulation(&grid, &bubble_params)?;
```

## Expected Output (if applicable)

The program prints physics background, collapse metrics, and emission/spectral comparisons for the single-bubble scenario.
