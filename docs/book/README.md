# Kwavers Ultrasound Physics Book

This directory is the source tree for the in-repository book on ultrasound physics for therapy, diagnostics, and theranostics. Chapters are split by application domain and reference the production modules that implement the corresponding models.

## Chapters

- [Therapy](therapy.md)
- [Diagnostics](diagnostics.md)
- [Theranostics](theranostics.md)
